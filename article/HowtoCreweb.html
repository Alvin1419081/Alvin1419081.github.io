使用 Github Page 制作一个网站

一、平台构建

  1. 打开 Github，点击”new“（建议关于浏览器自带翻译），备用地址 Github 备用（可能遭到浏览器拦截）

  2. 按照下图进行设置

    注意：名称务必设置为用户名，即和前面的名称一致，否则网站无法主动构建

  3. 创建第一个文件

    此图不是我的截图，不用管名称

  4. 此处创建的文件为网页的主页面，需要注意：

    1. 名称最好是 index.html，这样输入域名即可直接转到此页面，如我的个人网站主页：Alvin1419081.github.io，但是对于后新建的网页无法使用 index.html 名称，可采用任意名称。但输入网址时需要带上 html，比如我之前的论文网页或者我的同学录。
    2. 记住带上 html 后缀名，非常重要，因为网页本质上是文件。一般的大型网站后缀经过特殊处理，所以无需 html 后缀，但我们需要。
    3. 内容即网站的代码

  5. 在对话框中无需乱写直接提交，点击 Commit changes

  6. 网站默认开始构建，可以到 Action 选项卡中查看网站构建速度

  7. 网站构建完成后，转到 Settings>Pages，会看到有一个横幅，包含 Visit site 按钮，单击按钮即可转到你的网站

    注意：*.github.io 在中国大陆可能不定时无法访问

后续

如果需要构建更多网站，转到 库>Add File>Create new file 步骤与上述一致。

如果需要更改网站，转到 库>你的网站文件>工具栏中的”笔“图标，然后与上述步骤一致

注意：提交更改时对话框中的名称可能更改为 Change *.html，不用管它，也不要点下面的东西，直接 Commit Changes

如果网站中包含图片，可以转到 Add File>Upload files

如果需要文件夹整理，转到 Add File>Create new file / Upload Files 在路径栏中直接输入路径 文件夹名称/

注意：提交 html 文件后，文件夹会被自动创建

二、制作网站

AI 术语

我需要制作一个网站，它有着___________功能、UI 采用____________设计语言（建议为 Windows 11 UI 或 Material You 3），以及尽可能多的具体描述

使用模板创建（适合博客）

向 AI 发送制作好的 html 文件，或者提供一个网址，输入请你以________为网站模板，做一个新的博客网站，将文章内容替换为……

可以使用 Typedown 软件（从 Microsoft Store 获取）写文章，然后导出为 html，更方便。

推荐的 AI

Google Gemini：理解力更好，网站制作能力强，支持 Canvas。
